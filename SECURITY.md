# Security Policy

## Supported Versions

Security updates are provided for the latest published version of `@kripa006/404-ui`.

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

To report a vulnerability, use GitHub Security Advisories for this repository if available, or contact the maintainer listed in the package metadata. Include as much detail as possible:

- A clear description of the vulnerability
- Steps to reproduce the issue
- Affected package version or commit SHA
- Any known impact or workaround

We will acknowledge valid reports as soon as possible and coordinate a fix before public disclosure.

## Scope

This project is a UI component library. Security reports should focus on issues introduced by the library itself, such as unsafe DOM handling, vulnerable dependencies, or build/publishing risks.
